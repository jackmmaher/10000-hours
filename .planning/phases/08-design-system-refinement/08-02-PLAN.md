**Analysis context:**
Current typography violations:
- `text-[10px]` in Navigation.tsx - arbitrary value, too small
- 10+ font size variations across codebase
- Inconsistent line-height application
- Missing letter-spacing for section labels

**Target 5-style hierarchy:**
1. Display (72px) - Hero text, total hours
2. Heading (24px) - Section titles
3. Subheading (16px) - Card titles, labels
4. Body (14px) - Primary content
5. Caption (12px) - Metadata, hints, timestamps

**Skill guidance:**
- Body text: 14-16px, 400 weight, 1.5 line-height
- Never pure black (#000), never pure white (#fff) - already handled
- Left-align everything
- Letter-spacing: 0 for body, slight positive (0.5-1px) for ALL-CAPS labels
</context>
