How to Use These Wireframes

  1. Click any card above to view that screen's wireframe
  2. Within each wireframe, click buttons and links to navigate between screens
  3. Use the state tabs at the top of each screen to toggle between different states (e.g., Free vs Premium, Idle vs Running)
  4. Each page works standalone - you can bookmark individual screens
  5. These wireframes are responsive but optimized for iPhone viewport (375 x 812px)
